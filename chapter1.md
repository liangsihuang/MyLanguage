# Data Structure

### Map

c++ 称为容器 container

类似还有：Vector, Grid, Stack, Queue, Set

声明： Map&lt;int, ClassName&gt; theMap;

* key 的类型是 int , value 的类型是 某个 class

运算符被重载

* 某个元素：theMap\[ 'key' \]

遍历采用迭代器 iterator

* Map&lt;int&gt; :: Iterator

相当于 python 的 dict

### Vector

声明： Vector&lt;int&gt; theVector;

相当于 python 的 列表 list
